# Images Folder

Поместите сюда изображения для портфолио:

- `image1.png` - Первое изображение портфолио
- `image2.png` - Второе изображение портфолио
- `image3.png` - Третье изображение портфолио
- `image4.png` - Четвертое изображение портфолио

**Рекомендуемые размеры:** 500x400px или больше (сохранят пропорции)
**Формат:** PNG

После добавления изображений они автоматически отобразятся на странице `portfolio.html`.
